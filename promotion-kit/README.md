# Promotion Kit

半自动发布包：每个平台一篇定制文案，一键复制正文，一键打开发布页。  
建议人工检查后发布，遵守各平台版规和推广/开源项目分享规范。

## 用法

复制某个平台文案：

```powershell
.\promotion-kit\copy-post.ps1 juejin
```

打开常用发布页：

```powershell
.\promotion-kit\open-publish-pages.ps1
```

可用平台：

- `juejin`
- `zhihu`
- `v2ex`
- `linux-do`
- `segmentfault`
- `oschina`
- `csdn`
- `cnblogs`
- `appinn`

## 发布建议

- 每个平台只发一贴，不要短时间重复顶帖。
- 标题和正文可以按平台再微调。
- 技术社区优先讲设计过程和踩坑，不要只发“求 star”。
- 论坛类平台优先发到“分享、推广、开源项目、软件议题”等合适板块。
